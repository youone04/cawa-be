
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Teknologi version
1. package manager: yarn (version: 1.22.22)
2. node version: 20.0
3. nestjs version: 9.0
4. rdbms: mongose (version: 8.0)

## Project setup

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
